# 🚀 Dazk Website Rebuild

A modern, high-performance, and visually stunning web application for **Dazk Cloud Analytics**. This project is a complete rebuild focusing on premium design, modular architecture, and exceptional user experience. Built with React and Vite.

## 🌟 Key Features

- **Premium UI/UX:** Glassmorphism, dynamic gradients, and smooth animations.
- **Cloud Analytics Dashboard:** Detailed analysis reports with an intuitive interface.
- **High Performance:** Lightning-fast builds and module replacement via Vite.
- **Modular Components:** Structured and reusable React components for easy maintainability.
- **Responsive Design:** Completely fluid and mobile-optimized layouts out-of-the-box.

## 🛠️ Technology Stack

- **Core:** React 18, Vite
- **Styling:** Vanilla CSS (Advanced CSS3, CSS Variables)
- **Routing:** React Router v6

## 📁 Project Structure

```text
dazk-website-rebuild/
├── public/                 # Static assets
└── src/
    ├── assets/             # Images, fonts, and global icons
    ├── components/         # Reusable UI components (Navigation, CaseStudies, etc.)
    ├── pages/              # Route-level components (LandingPage, Analyze, etc.)
    ├── App.jsx             # Root component
    ├── index.css           # Global design system & utility classes
    └── main.jsx            # Application entry point
```

## 🚀 Getting Started

### Prerequisites

You need [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd "Dazk Website Rebuild"
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173`.

## 📦 Build for Production

To create a production-ready build, run:
```bash
npm run build
```
The optimized bundle will be generated in the `dist` directory. To preview it locally, run:
```bash
npm run preview
```
